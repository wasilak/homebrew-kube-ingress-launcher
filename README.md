# Homebrew Tap for Kube Ingress Launcher

This is the official Homebrew tap for [Kube Ingress Launcher](https://github.com/wasilak/kube-ingress-launcher).

## Installation

```bash
brew tap wasilak/kube-ingress-launcher
brew install --cask kube-ingress-launcher
```

## Updating

```bash
brew upgrade --cask kube-ingress-launcher
```

## Uninstalling

```bash
brew uninstall --cask kube-ingress-launcher
```

## About

Kube Ingress Launcher is a standalone Tauri-based macOS desktop application that provides Spotlight-like search for Kubernetes ingress resources.

For more information, visit the [main repository](https://github.com/wasilak/kube-ingress-launcher).
